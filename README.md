# High-Quality-Invoice-Images-for-OCR
0**"High-Quality Invoice Images for OCR"** mô phỏng quy trình tự động hóa xử lý dữ liệu hóa đơn (Invoice / Order) trong doanh nghiệp.   Mục tiêu chính là **thu thập, tiền xử lý, lưu trữ, và trực quan hóa dữ liệu hóa đơn** để hỗ trợ các hệ thống OCR, AI hoặc phân tích tài chính.
